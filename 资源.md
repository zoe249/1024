## 各站点建议搜法

### itch.io

- `mobile puzzle ui`
- `casual ui pack`
- `puzzle sound pack`

### Kenney

适合先拿基础 UI、按钮、简单音效做第一版。

### OpenGameArt

适合找免费贴图和特效素材，但要仔细看授权。

### Pixabay

适合搜：

- `click`
- `pop`
- `chime`
- `impact`

### Freesound

适合找单个质量更细的音效，但风格统一需要自己筛。

## 筛选标准

### 图片资源

- 风格统一
- 分辨率不要太低
- 优先透明背景 PNG
- 尽量同一套来自同一个作者
- 移动端竖屏适配友好

### 音效资源

- 长度短
- 起音干净
- 不要混响太重
- 同一套音色尽量一致
- 适合移动端短反馈

## 授权检查

下载前确认：

- 是否允许商用
- 是否要求署名
- 是否允许修改
- 是否允许上架应用商店

不要只看“free”，要看具体 license。

## 推荐优先级

建议按以下顺序替换资源：

1. 棋子底图/棋子风格
2. 背景和棋盘底板
3. 按钮与 UI
4. 合成与点击音效
5. 特效贴图
6. BGM

## 免费优先方案

如果想先快速搭出一版：

1. 用 `Kenney` 找一套基础 UI
2. 用 `itch.io` 或 `OpenGameArt` 找棋盘/背景
3. 用 `Pixabay` + `Freesound` 找按钮、合成、失败音效
4. 用 `OpenGameArt` 找 glow / burst 类特效贴图
